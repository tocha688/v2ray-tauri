# v2rayTA（开发中）

v2rayTA是基于tauri制作的v2ray gui客户端

目前支持：socks、shadowsocks、vmess

本地代理端口:
- socks：10818
- http/https：10819

目前只支持windows，其他系统等功能都开发差不多了在进行适配吧。
